(Using JavaScript) - Create a new HTML-element inside a"div". Hide an existing element ("button")

USING JavaScript: Listen for the event "load" on the window and call a GENERIC function.
The function will look in the document for the ID "btn", and add a "click"-event to it.
When this "btn" is clicked, call GENERIC function to:
- Look in the document and get the element by ID "output" and create a new HTML-element with text "Geklikt".
- Also, look in the document and get the element by id "btn" and change the display style to "none".
