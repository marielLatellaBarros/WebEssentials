Bind (as of jQuery 3.0 deprecated. Better .on() ) - Bind click, dblclick, mouseenter mouseleave, toggleClass

When the document is ready call function.
1. Attach ("bind") a handler to a "click"-event for the element "p" and get the coordinates of X and Y.
Then attach these coordinates to the "span" element.
2. Attache ("bind") a handler to a "dblclick"-event for the element "p" and when a double click happens, the text in the "span" element changes to "Double-click happened".
3. Attache ("bind") a handler to a "mouseenter mouseleave"-event for the element "p" and when the mouse enters or leaves the paragraph, the background in this element (so, "p") "toggles a Class" to a different color. in the "span" element changes to "Double-click happened".
