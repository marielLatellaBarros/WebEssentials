Fadeout and FadeIn

When the document is ready, call generic function.
Listen for a "click-event" for ID "#div1" and when this happens for "this" #div1 fade out at 3000 milliseconds (3 seconds) AND call another generic function that looks at ID "#div2" (initially display: "none") that fades in at 3000 milliseconds.
Note: - default values: [duration: 400 milliseconds], [easing: "swing"], [complete].
- 'fast' (200 milliseconds) and 'slow' (600 miliseconds).
- Once the opacity is 0 (100% see-through), the display is set to: none.
- FadeIn() sets the css display from "none" to "block".
