animate - opacity, left and height ("toggle")

Both divs are positioned relative to one another. Same width and same heigt = square.
When the document is ready, call generic function and listen for a click event on ID "#btn1".
When a click happens, the "#div1" will animate!. Oppacity is set to "toggle", left is set to +=20, and height is also set to "toggle". All at a speed of 1000 miliseconds.
When "#btn2" is clicked, then animate "#div2". The opacity is set to 0, left to +=100 and height also "toggle", all with a speed of 1000 as well.
- ALSO, when #btn2" is clicked, call anther function that will animate "this" object (#div2) and will set the opacity to 1, it will pull it back with left -=100 and the hight will also toggle, all at the same speed of 1000.
