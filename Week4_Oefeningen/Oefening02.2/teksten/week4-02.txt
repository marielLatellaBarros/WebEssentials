Competentiebeschrijving
Je leert de nodige vaardigheden om zelf een statische website te maken. Hiervoor werken we met de gangbare
webstandaarden, namelijk Hypertext Markup Language (HTML) en Cascading Style Sheets (CSS). In het laatste deel
krijg je een introductie tot meer geavanceerde standaarden en scriptingtalen om zo je eerste dynamische pagina's
te bekomen. Het is de bedoeling dat je naast het hanteren van deze vaardigheden ook een gefundeerde keuze kan
maken aan de hand van het doel van de website of onderdelen van de website. Je leert ook je code te
documenteren, te onderhouden en te hergebruiken. Je verwerft vaardigheden om multimediatoepassingen te
realiseren in verschillende contexten. Om je de nodige inzichten te geven in de manier waarop dit alles werkt,
zal je rechtstreeks met de code werken door middel van een tekst editor. Je leert ook hoe je een website kan
installeren en valideren.

Evaluatie
Je kent en begrijpt de concepten van de meest gebruikte webtechnologie&euml;n zoals Hypertext Markup Language
(HTML, xHTML), Cascading Style Sheets (CSS), Javascript en het DOM. Je kan een gefundeerde keuze maken voor de
opmaak van een statische website. Je kan een webpagina volledig coderen, documenteren en onderhouden. Je kan
koppelingen en afbeeldingen invoegen, lijsten en tabellen gebruiken en formulieren opmaken. Voor de lay-out van
een website kan je de verschillende methodes toepassen, zoals frames, tabellen en layers. Je kan eenvoudige
scripts (JavaScript) ontwikkelen. Je begrijpt het Document Object Model en kan dit toepassen in eenvoudige
toepassingen. Je kan een website valideren en installeren.
